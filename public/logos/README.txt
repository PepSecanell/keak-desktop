Drop brand logo PNGs here. Keak uses public/logos/<id>.png before the Simple Icons CDN,
so a file here overrides everything. Use these exact filenames (lowercase, .png):

  google.png  microsoft.png  slack.png  figma.png  gumloop.png
  heygen.png  gamma.png  higgsfield.png  manus.png

Others already resolve from the CDN (notion, supabase, github, shopify, telegram,
perplexity, elevenlabs, zapier, n8n, resend) — add a PNG here to override any of them too.
Square-ish logos look best. Transparent background is fine.
